# SCS-Website-Cybersecurity
SCS cybersecurity program

## Web structure

- `web/cs_project_chinese.html` is the public student and family Chinese edition.
- `web/cs_project_english.html` is the public student and family English edition.
- `web/teacher_chinese.html` 
- `web/teacher_english.html` are the internal teacher edition with teaching tips, resources, assessment, and DOCX source tables.
- `web/resource_chinese.html` 
- `web/resource_english.html` are the video lib.
- `web/assets/data/` stores reusable unit data and extracted DOCX table data.
- `web/assets/scripts/` stores page-specific JavaScript and shared UI helpers.
- `web/assets/styles.css` stores the shared visual system for both editions.
